Etiquetas visibles en HTML

Etiquetas utilizadas en texto

< h1 a h6 >

Son los títulos de la página, están divididos en 6 tipos, siendo el h1 el más importante y el principal, y los demás siguen una jerarquía de importancia. Es importante porque ayuda a administrar cada encabezado y que el contenido esté estructurado y organizado.

< p >

Es utilizado para escribir párrafos, de esta manera se pueden agrupar y estructurar el contenido que sea de texto en párrafos separados.

< strong >

Se utiliza para colocar en negrita y un texto y así resaltarlo. Ejemplo: Hola.

< em >

Se usa para colocar en cursiva y un texto y así resaltarlo. Ejemplo: Hola.

< span >

Es un elemento contenedor en línea genérico que realmente no tiene un efecto visual por sí solo, pero se utiliza para aplicar estilos al texto o para agrupar elementos en línea con algunos atributos.

< mark >

Resalta partes de un texto, se utilizado para llamar la atención del usuario hacia una parte específica del contenido.

< abbr >

Se utiliza al momento de abreviar palabras largas en el documento.

< fipcation >

Define el título para una figura "figure".

< legend >

Define un título para los elementos "fieldset", "figure", "details".

< small >

Se utiliza principalmente para fines de estilo y presentación. No debe confundirse con el uso de la etiqueta "small" para indicar contenido legalmente significativo, como las notas y los avisos legales, ya que este último caso se considera obsoleto en HTML5 y se recomienda utilizar elementos más específicos, como "footer" o "aside", junto con estilos CSS.

< sub >

Se utiliza para mostrar texto en formato de subíndice, es decir, un texto que aparece ligeramente más abajo y más pequeño que el texto circundante. Se utiliza comúnmente para mostrar fórmulas químicas, ecuaciones matemáticas, números ordinales y otras notaciones que requieren subíndices.

Etiquetas que interactúan en el sitio web

< br >

Etiqueta que se usa para crear un salto de línea en el texto. Es el clásico "enter que se da después de un punto y aparte".

< hr >

Se usa comúnmente como un separador ya que crea una línea horizontal.

< a >

Se utiliza para agregar enlaces o hipervínculos, lo que permite a los usuarios redirigirse a otra pagina web o a otra sección de la misma página.

< iframe >

Incrusta contenido de otros documentos HTML dentro de la misma página web.

< time >

Etiqueta que representa una fecha, hora o un período de tiempo. Por ejemplo:
Tengo que salir mañana a las 2:00 PM

< table >

Se utiliza para crear una tabla, todos los elementos de la misma deben ir siempre entre estas dos etiquetas.

< tr >

Esta etiqueta es la que se utiliza para indicar una fila en una tabla.

< td >

Define la celda de una tabla.

< th >

Es la etiqueta que permite definir el encabezado que tendrá una celda.

< border >

Ejemplifica el borde de un cuadro o una tabla, y se le debe indicar como atributo a la etiqueta table. A continuación se realizará una tabla utilizando 4 etiquetas mencionadas anteriormente:

< colgroup >

Especifica un grupo de una o más columnas en una tabla para aplicarles propiedades de formato (como el color).

< tfoot >

Define el pie de una tabla.

< tbody >

Representa la sección principal de la tabla que contiene las filas de datos.

< thead >

Define el encabezado que tendrá una tabla.

< form >

Es utilizada para crear un formulario.

< input >

Se utiliza para crear controles interactivos de entrada en un formulario, esto permite a los usuarios ingresar y enviar datos. Es importante mencionar que existen muchos tipos de inputs, como lo son texto, email, password, number entre otros.

< button >

Esta etiqueta crea un botón interactivo, se suele usar en formularios.

< label >

Se utiliza para definir el nombre de un elemento de control del formulario (como un botón).

< textarea >

Es una etiqueta donde se coloca un campo donde el usuario puede agregar texto.

< select >

Es utilizado para crear un menú desplegable que permite a los usuarios seleccionar una opción de una lista de opciones predefinidas.
