Etiquetas no visibles en HTML

Contenedores

< !DOCTYPE Html >

Indica el tipo de documento HTML que se está utilizando.

< html >

Es la raíz del documento, todo el documento debe de estar dentro de las etiquetas.

< head >

Define la información meta sobre el documento, como el título de la página, enlaces a hojas de estilo externas o scripts, y otros metadatos.

< header >

Define el encabezado de un documento, se utiliza para colocar contenido relacionado con la introducción, como el título de la página, el logotipo, la navegación principal y otra información que sea relevante.

< body >

Todo el contenido de la pagina web debe de estar dentro de las etiquetas para mostrarse.

< div >

Es utilizado para crear divisiones o secciones, es como un contenedor de bloque que agrupa y organiza elementos HTML dentro de él.

< article >

Define el contenido independiente y autónomo dentro de un documento.

< aside >

Aloja contenido relacionado con el principal de la página, pero que es considerado como información complementaria.

< nav >

Define una sección de navegación en un documento web.

< footer >

Define el pie de página del documento, que puede constar de información sobre el autor, derechos de autor, detalles de contacto y enlaces a documentos o recursos relacionados.

< embed >

Incrusta el contenedor de una aplicación externa (no html), como un complemento del navegador.

< figure >

Agrupa contenido multimedia, como imágenes, gráficos o videos, junto con su leyenda o descripción.

Metadatos o datos internos

< link >

Enlaza un archivo externo con la página HTML actual, se utiliza principalmente para enlazar hojas de estilo CSS, pero también se puede utilizar para enlazar otros tipos de archivos.

< meta >

Proporciona información adicional sobre una página web, como definir el autor del sitio, o la descripción del mismo.

< style >

Define estilos de CSS dentro de un documento HTML, con esta etiqueta se especifican reglas de estilo que se aplicarán a elementos específicos en la página.

< script >

Puede incluir y ejecutar código de JavaScript en una página web. Puede contener declaraciones de script directamente o enlazar a un archivo externo que contenga el código de JavaScript.

< noscript >

Se usa para proporcionar contenido alternativo en el caso de que el navegador del usuario no admita JavaScript o cuando el JavaScript esté desactivado en el navegador.

< base>

Especifica la URL base para todas las URL relativas en una página web.
